The easiest way to beautifully create and display your wedding registry.


== Description ==

Wedding Registry is a complete registry solution for adding registry links in a light, performant, and easy to use plugin. Add popular registry shortcodes to your WordPress themes via the WordPress visual editor. 

Check out the [live demo](http://demo.themebeans.com/emma/registry/ "Live demo of the Wedding Registry WordPress plugin") of the plugin in action on the [Emma WordPress theme](http://demo.themebeans.com/emma/ "Emma WordPress Theme") by the [ThemeBeans WordPress theme shop](http://themebeans.com/ "ThemeBeans - Premium WordPress Themes").

= Example Usage: =

The following is an example shortcode with the "Etsy" shortcode, inserted via the WordPress visual editor.

[registry info="macys" url="http://your-registry-url"]


== Installation ==

1. Upload the `wedding-registry` folder to your `/wp-content/plugins/` directory or alternatively upload the wedding-registry.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your local computer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add shortcodes to content areas via the dropdown item on the visual editor.


== Frequently Asked Questions ==

= How do I add a shortcode?  =

You'll see a new button at the end of the default WordPress visual editor that states `Insert Registry Shortcode` when hovered. Select a registry location from the dropdown and insert your URL in the `url=""` shortcode parameter.
